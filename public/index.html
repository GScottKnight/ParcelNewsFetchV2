Parcel News Pipeline

Live site placeholder. This app ingests parcel-carrier news, classifies relevance, extracts cost-impact levers, and dedupes into canonical events for downstream calculators.

Build Phases

  1. Foundations: shared enums/schemas and config.
  2. Ingestion & Storage: fetch + persist raw articles.
  3. Stage 1 Relevance: cheap classifier.
  4. Stage 2 Extraction: structured lever extraction + signature.
  5. Event Dedupe: canonical event merge.
  6. Calculator Interface & Ops.

See repository README for details and the spec in Documents/parcel_news_pipeline_spec (1).md.
